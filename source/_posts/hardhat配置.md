首先打开hardhat的[官网](https://hardhat.org/)
<!-- more -->

初始化package.json
```
yarn init -y
```
添加hardhat
```
yarn add --dev hardhat
```
初始化一个hardhat框架
```
npx hardhat init
```
这时候会提示新建哪种项目，选择typescripte框架，等待下载完成
